## 🛠️ Prototyping on CASES-99 (Julia + Chebyshev)
Because boundary-layer data is discretely sampled across tower heights (z_i), standard finite difference methods will introduce catastrophic noise when taking the third derivative for Jerk Energy (E_j).
Your choice of **Chebyshev pseudospectral methods** is computationally elegant here. By mapping your physical domain z \in [0, h] to the Chebyshev domain \xi \in [-1, 1], you can leverage analytical differentiation matrices (D) to evaluate these integrals to machine precision.
Here is a structurally sound Julia prototype designed to ingest a observed profile, map it onto a Chebyshev grid, evaluate its geometric invariants via matrix-vector multiplication, and return the shape energy feature vector:
```julia
using LinearAlgebra
using FFTW

struct ChebyshevProfile
    z_nodes::Vector{Float64} # Physical heights
    values::Vector{Float64}  # Profile values (e.g., Potential Temperature θ)
    D1::Matrix{Float64}      # First differentiation matrix
    D2::Matrix{Float64}      # Second differentiation matrix
    D3::Matrix{Float64}      # Third differentiation matrix
    weights::Vector{Float64} # Clenshaw-Curtis quadrature weights for integration
end

"""
    compute_shape_energy(profile::ChebyshevProfile)

Computes the geometric shape energy vector for an atmospheric profile.
"""
function compute_shape_energy(profile::ChebyshevProfile)
    f  = profile.values
    w  = profile.weights

    # Compute analytical derivatives via matrix-vector multiplication
    f′  = profile.D1 * f
    f″  = profile.D2 * f
    f′′′ = profile.D3 * f

    # 1. Gradient Energy
    E_g = dot(w, f′ .^ 2)

    # 2. Exact Curvature Energy
    κ = abs.(f″) ./ ((1.0 .+ f′ .^ 2) .^ 1.5)
    E_κ = dot(w, κ .^ 2)

    # 3. Jerk Energy (The Brittleness Metric)
    E_j = dot(w, f′′′ .^ 2)

    # 4. Spectral Roughness via FFT (assuming uniform spatial perturbation)
    # Alternatively, use the decay rate of the Chebyshev coefficients directly!
    cheb_coefs = profile.D1 \ f # Linear solve to find spectral expansion weights
    N = length(cheb_coefs)
    R = sum(((1:N) .^ 2) .* (abs2.(cheb_coefs)))

    return (E_g = E_g, E_κ = E_κ, E_j = E_j, R = R)
end

```
